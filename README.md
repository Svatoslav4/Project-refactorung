📌 Опис проєкту

Цей проєкт присвячений рефакторингу програмного коду. Основна мета — покращити якість коду без зміни його функціональності.

Було проаналізовано вихідний код, виявлено «запахи коду» та застосовано різні техніки рефакторингу для підвищення читабельності, підтримуваності та структури.


⚙️ Використані технології

* Python 3
* unittest (для тестування)

🔧 Основні зміни

* Покращено іменування змінних і методів
* Видалено дубльований код
* Розбито великий код на менші методи
* Додано класи для кращої структури
* Замінено "магічні числа" на константи
* Спрощено умовні конструкції


🧪 Тестування

Було створено 15 модульних тестів, які перевіряють:

базову логіку обчислень
граничні випадки
обробку некоректних даних


▶️ Як запустити проєкт

1. Відкрити термінал у папці проєкту
2. Виконати команду:

```
py -m unittest tests/test_cases.py
```
